== Description ==

muestra el estado del clima de la ciudad que elijas utilizando la api de yahoo! weather, se implementa como funcion en la plantilla, shortcode o widge

== Installation ==

Descomprime el archivo zip y colocalo en el directorio wp-content/plugins/ de tu instalacion de wordpress y despues activa el plugin en la pagina de plugins, es posible implemenarlo agregando la funcion clima\_freekitime() en la parte de la plantilla que la desees, tambien es posible implementarlo como shortcode [clima\_freekitime] y por ultimo cuenta con su widget.
